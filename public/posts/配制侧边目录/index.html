配制侧边目录

2026-02-03 · 1 min · map[email: name:tdx]

以下是配置文件

调整侧边目录位置
修改 CSS 文件

notepad assets/css/extended/custom.css
删除全部内容，粘贴以下代码：

CSS

/* ========================================
   隐藏原有顶部目录
   ======================================== */
.post-single > .toc {
    display: none !important;
}

/* ========================================
   右侧固定目录 - 移到最右边
   ======================================== */
#TableOfContents {
    position: fixed !important;
    right: 10px !important;
    top: 80px !important;
    width: 220px !important;
    max-height: 75vh !important;
    overflow-y: auto !important;
    background: var(--entry) !important;
    padding: 15px !important;
    border-radius: 10px !important;
    box-shadow: 0 2px 15px rgba(0,0,0,0.1) !important;
    z-index: 999 !important;
    font-size: 13px !important;
}

#TableOfContents::before {
    content: "📑 目录";
    display: block;
    font-size: 16px;
    font-weight: bold;
    color: var(--primary);
    padding-bottom: 10px;
    margin-bottom: 10px;
    border-bottom: 2px solid var(--primary);
}

#TableOfContents ul {
    list-style: none !important;
    padding: 0 !important;
    margin: 0 !important;
}

#TableOfContents ul ul {
    padding-left: 12px !important;
    border-left: 2px solid var(--border) !important;
    margin-left: 6px !important;
}

#TableOfContents li {
    margin: 5px 0 !important;
}

#TableOfContents a {
    display: block !important;
    padding: 6px 10px !important;
    color: var(--secondary) !important;
    text-decoration: none !important;
    border-radius: 5px !important;
    font-size: 13px !important;
    line-height: 1.4 !important;
    transition: all 0.3s !important;
}

#TableOfContents a:hover {
    background: var(--primary) !important;
    color: white !important;
}

/* ========================================
   文章内容区域 - 不受影响
   ======================================== */
.post-content {
    max-width: 100% !important;
    margin-right: 0 !important;
}

.main {
    max-width: 800px !important;
    margin: 0 auto !important;
}

/* ========================================
   滚动条美化
   ======================================== */
#TableOfContents::-webkit-scrollbar {
    width: 4px;
}

#TableOfContents::-webkit-scrollbar-thumb {
    background: var(--primary);
    border-radius: 4px;
}

/* ========================================
   小屏幕隐藏目录
   ======================================== */
@media (max-width: 1400px) {
    #TableOfContents {
        display: none !important;
    }
}

/* ========================================
   暗色模式
   ======================================== */
.dark #TableOfContents {
    background: #1e1e1e !important;
    box-shadow: 0 2px 15px rgba(0,0,0,0.4) !important;
}
重启服务
Bash

rm -rf public resources
hugo server -D
